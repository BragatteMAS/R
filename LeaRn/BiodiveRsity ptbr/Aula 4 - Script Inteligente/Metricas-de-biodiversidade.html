Mtricas essenciais de Biodiversidade

Um script criado pelo Prof Marcos Vital, para a Academia do R

Este script foi escrito usando R Markdown no RStudio verso 1.4.1106, e testado em R verso 4.1.0.

O que este script?

Este script foi criado para que qualquer pessoa possa realizar uma avaliao automtica de diversas mtricas bsicas (mas essenciais) tipicamente usadas em anlises de Biodiversidade.

A maneira mais fcil de usar, especialmente se voc for iniciante no R, salvar os seus dados em um arquivo com o nome dadosbio.csv e salv-lo na pasta de trabalho que estiver sendo usada no R.

Voc tambm ir precisar do arquivo curvacum.R, que carrega uma funo personalizada para criao do grfico de curva de acmulo de espcies.

As instrues detalhadas de como us-lo foram exibidas na ltima aula da Maratona de Anlise de Biodiversidade no R, que foi promovida pela Academia do R

Na sua primeira tentativa de utilizar este script, recomendo fortemente que use o arquivo de dados de exemplo que foi fornecido junto com ele durante a aula. Depois de conseguir fazer tudo com o exemplo, tente com os seus prprios dados! ;)

Para que o script funcione, voc deve ter o pacote vegan instalado, alm do pacote RMarkdown e todas as suas dependncias.

Lendo os dados

Os seus dados devem estar em formato csv, separado por vrgulas. Caso esteja com separao em ponto e vrgula, veja a explicao aps o comando para alterar.

O nome do arquivo deve ser dadosbio.csv, a no ser que voc prefira alterar o comando a seguir, ok? E no se esquea de conferir se ele est salvo na pasta de trabalho que estiver sendo usada pelo R no momento.

A estrutura dos dados neste arquivo deve, obrigatoriamente, seguir o padro unidades amostrais nas linhas, espcies nas colunas. Na dvida, veja o arquivo de exemplo.

  • Se o seu arquivo usar ponto e vrgula como separador de colunas, mude sep para ;
  • Se os seus dados usarem separador decimal, use ponto como separador, ou altered dec para , caso use vrgulas.

Conferindo rapidemente o tamanho do arquivo: ele tem 30 linhas e 43 colunas.

Se a descrio acima no estiver correta, confira o arquivo, seu formato, o local que est salvo, e comece de novo, ok?

Calculando alguns ndices de diversidade

Vamos trabalhar com dois ndices amplamente usados: o ndice de Shannon e o ndice de Simpson. Para ambos, vamos calcular o valor total para toda a amostra, e o valor mdio (com seu desvio padro) para cada unidade amostral.

Os valores de Simpson sero apresentados na forma original, variando de 0 a 1, na qual valores mais baixos indicam maior diversidade (este formato eventualmente chamado de ndice de dominncia de Simpson).

ndice de Shannon

  • O ndice de diversidade de Shannon da amostra completa de 3.2520821.
  • O valor mdio para cada unidade amostral de 2.8067941, com desvio padro de 0.1298341

ndice de Simpson

  • O ndice de diversidade de Shannon da amostra completa de 0.0523291.
  • O valor mdio para cada unidade amostral de 0.0727365, com desvio padro de 0.0098037
